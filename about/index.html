About me

Projects

Fully Reversed Engineering 2012-2013

Developed a low-cost, high-resolution 3D scanner. Researched existing methods for obtaining scan information. Worked as part of a team to derive equations needed for the application. Created a new method of 3D scanning to allow our system to be constructed at low cost. Implemented calculation and user interface software using C++, OpenCV, and Qt. Wrote paper and created poster to communicate methods and findings.

Scout Transatlantic 2012-2013

Designed and wrote communications software for an autonomous robotic boat that ended up (unofficially) breaking the world record for unaided robotic boat navigation. Created an Arduino library for interfacing with an Iridium satellite transmitter. Wrote a Python cron task to handle incoming satellite transmissions on a web server and push them to a MySQL database. Built an interactive PHP/HTML/JavaScript frontend based on a team member's visual design.

Building Blocks of Venice 2011

Researched and documented maintenance techniques for elements of Venice's infrastructure. Participated in professional sponsor meetings. Created over 40 illustrations. Developed a Java application along with PHP API to easily convert database entries into MediaWiki pages.
